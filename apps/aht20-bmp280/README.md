# AHT20 + BMP280 传感器测试工程 (ESP32-S3)

本工程用于在 ESP32-S3 上通过 I2C 同时读取 AHT20（温湿度）和 BMP280（气压/温度）传感器的数据。

## 硬件接线

两个传感器共享同一条 I2C 总线：

| 传感器信号 | ESP32-S3  | 备注       |
| :--------: | :-------: | :--------- |
|  **VCC**   | **3.3V**  | 传感器电源 |
|  **GND**   |  **GND**  | 共地       |
|  **SDA**   | **GPIO1** | I2C 数据线 |
|  **SCL**   | **GPIO0** | I2C 时钟线 |

## 快速运行

在工作区根目录执行：

```powershell
cargo run -p aht20-bmp280-app --release -- --port COMx
```

## 技术细节

- **I2C 共享**：使用 `shared-bus` 库实现 I2C 句柄的并发访问（适配 `no_std`）。
- **AHT20**：使用 `embedded-aht20` 驱动，测量高精度温湿度。
- **BMP280**：使用 `bme280` 驱动（支持 BMP280 模式），测量大气压强。
- **驱动标准**：所有驱动均适配 `embedded-hal 1.0` 稳定版。

## 提示

- 如果 BMP280 无法初始化，请检查 SDO 引脚的电平状态。`BME280::new_primary` 默认地址为 `0x77`。如果模块地址是 `0x76`，请改为 `new_secondary`。
