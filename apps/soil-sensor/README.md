# 电容式土壤湿度传感器测试工程 (ESP32-S3)

这是一个基于 `esp-hal` 的 Rust 嵌入式工程，用于在 ESP32-S3 上通过 ADC (模数转换) 读取**电容式土壤湿度传感器**的电压数据，来判断土壤干湿情况。

## 硬件接线

传感器输出电压范围为 `0 ~ 3.0 VDC`，供电使用 `3.3 VDC`，非常适合直接连接到 ESP32 系列的 ADC 管脚。
本工程默认使用 `GPIO3` (ADC1 Channel 2) 进行电压测量：

| 土壤传感器 | ESP32-S3 | 备注 |
| :---: | :---: | :--- |
| **VCC** | **3.3V** | 传感器提供稳压，需接入3.3V电源 |
| **GND** | **GND** | 共地 |
| **AOUT** | **GPIO3** | 模拟信号输出，输入ESP32的ADC引脚 |

> **注意：** ESP32-S3 的 ADC 输入最高支持到 3.3V（需要设置为 `11dB` 衰减，本工程中已经配置为 `Attenuation::_11dB`）。如果您的引脚变更，请去对应源码 `src/main.rs` 内修改为对应的 `GPIO` 管脚即可。

## 快速运行

进入工作区根目录 (`esp-rs-lab`) 并运行以下命令编译和烧录：

```bash
cargo run --bin soil-sensor
```

运行后，您应该能在串口（monitor）中看到 ADC raw 计数和相对应的估算电压值（毫伏）：
```text
INFO - Initializing Soil Moisture Sensor on ADC (GPIO3)...
INFO - ADC Value: 1850 | Approx Voltage: 1490 mV
INFO - ADC Value: 2431 | Approx Voltage: 1958 mV
...
```

## 说明
电容式土壤湿度传感器在：
- 完全暴露于空气中时（**极干**），输出电压最大，ADC 读数也**最高**。
- 浸没在水里或湿润土壤中时（**极湿**），输出电压下降，ADC 读数也**最低**。
- 您需要在空载空气中记录一个最大干旱阈值，然后在水里记录一个最大湿润情况的阈值，随后根据您的绿植适宜区间映射转换成百分比形式（代码默认只打印基础数据，后续扩展请自行标定算法）。
